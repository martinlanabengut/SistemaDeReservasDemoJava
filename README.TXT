#Sistema de Reserva de Salas y salones

Requerimientos y herramientas utilizadas:

Se utilizó el IDE NETBEANS 11.1 Y lenguaje de programación JAVA 8 para
el desarrollo del código.

Para el modelamiento entidad-relación de la base de datos se utilizó 
MySQL Workbench 8.0-

Como servidor Apache y MySQL se utilizó Xampp(adjuntado el instalador
 en complementos).
